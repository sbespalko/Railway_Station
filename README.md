 На вокзал прибывают-отбывают поезда. Ведется журнал (время прибытия-отбытия (1-24)).<br>
 Необходимо узнать, какое максимальное кол-во поездов было на вокзале одновременно.<br>
 Также необходимо указать первый период, в который был максимум.<br>
 Вокзал может принять неограниченное кол-во поездов.<br>
